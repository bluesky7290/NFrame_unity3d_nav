This directory contains precompiled libraries.

dotNet

    Contains libraries for .NET-only users. (Not compatible with Unity.)

u3d

    Contains the primary Unity libraries.

u3d-extras

    Contains some optional NNMGen extras for Unity.  Includes components
    for annotating areas and off-mesh connections in a scene.

u3d-plugins

   The mandatory Unity plug-ins.

Note:

Unity users will usually import the Unity packages rather than use
these libraries.
